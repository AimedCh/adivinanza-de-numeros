# Juego de Adivinanza de Números 🎯🔢

Juego en el que debes adivinar un número entre 1 y 100, con pistas sobre si el número es mayor o menor.

## Tecnologías utilizadas 🛠️

- **HTML** 📝
- **Tailwind CSS** 🎨
- **JavaScript** 💻
